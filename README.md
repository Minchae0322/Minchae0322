## About

안녕하세요.

Java와 Spring Boot를 중심으로 백엔드 서비스를 개발하고 있습니다.

서비스를 구현하는 것에서 끝나지 않고, 운영 과정에서 발생하는 문제를 분석하고 더 나은 구조로 개선하는 과정을 즐깁니다.

최근에는 Kubernetes 기반 MSA, Event-Driven Architecture, Observability, 성능 최적화에 관심을 가지고 직접 서비스를 만들며 학습하고 있습니다.
